# USAGE
```bash
todome
```

# INSTALLATION

Just run:
```bash
make
```
The binary will be located at ~/.local/bin/todome .

# DE-INSTALLATION

Just run:
```bash
make deinstall
```
The binary will removed from ~/.local/bin/todome .
